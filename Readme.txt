Avant de compiler, choisissez la taille du tableau que vous voulez trier à la ligne 61 du main.c : int tailleTab = ?;
-> Remplacez ? par un nombre
ATTENTION : en fonction de votre ordinateur, la taille maximale du tableau peut varier.
Un tableau trop imposant peut donc causer une erreur de segmentation !

Pour compiler : utiliser la commande make dans le répertoire contenant les fichiers source.
Pour lancer le programme : ./main

Suivez ensuite les instructions à l'écran :
- Sélectionnez le ou les algorithmes de tri que vous voulez
- Sélectionnez le type de tri que vous souhaitez obtenir

-> Les résultats des temps d'exécution de chaque algorithme apparaissent !
